Datos y copias

Tus pensamientos se guardan solo en este dispositivo (en este navegador). Exporta a Markdown de vez en cuando y guárdalo en tu Vault: es tu copia duradera.

Para tenerla como app: abre el menú del navegador y elige «Instalar» o «Añadir a pantalla de inicio».

Planes con IA

Con una clave de API de OpenAI, la app puede proponer un siguiente paso para cada pensamiento. La clave se guarda solo en este navegador y tus pensamientos van directos de tu navegador a OpenAI — sin servidores intermedios.
